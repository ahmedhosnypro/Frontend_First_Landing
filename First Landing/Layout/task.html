Description

Let's start by setting the basic layout of your landing page.

First, you need to divide the HTML body into 3 parts: header, main, and footer. The header tag contains the navbar, the main tag comprises the content, and the footer tag, as you can guess, has the footer. For now, let's write the placeholder text inside these tags so that you can identify them.

Second, create a new CSS file named style.css and link it with the HTML. Next, select the body and set the maximum height at 100% of the visible screen area. Also, get rid of the default margins, set the height of the lines at 1.5, and change the default font-family to sans-serif.

Select the header, main, and footer and set their minimum height at 10%, 60%, 30% of the visible screen area, respectively. Additionally, give them a background color to set them apart.

Objectives

In this stage your website should have:

  • The header, main and footer tags in the HTML body with the order mentioned here;
  • A placeholder text inside these tags;
  • A link to a new CSS file;
  • Stick to the style rules for tags as described above.
